Graph Planarity Tester

Description: Solely developed a web-based app to graphically create and manipulate graphs. The core defining feature is the ability to test whether or not a graph is planar. If the graph is planar the app will redraw the graph as a planar embedding, otherwise it will show the Kurotowski minor.

Link: jandreou.com/graph Note: Best experience on desktop!

Beugo the Blob

Description: Created an Android game where you chase a ghost as a bouncing blob. The player controls how high the blob bounces to dodge randomly generated incoming obstacles. The soft body physics of a blob are emulated using a spring mass system compressing the inner portions of the body on impact and a triangle fanning algorithm. Additional features include different difficulties, interactive tutorial, level system and much more.

Developed in: Java, OpenGL, XML, Android

Link: play.google.com/store/apps/beugo Note: If link doesn't work search 'beugo' on google play.
